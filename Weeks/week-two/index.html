Alexa Skills Development


Intents


Amazon Alexas intents are functions attempts to fulfill end user's queries.Intents can have positional arguments called slots that to map (sort of a like HashMaps, or Linked Lists) to improve the intents capabilities.


Utterances

Utterances are snip bits of end-user responses so that the skill can know how to respond.


Multimodal Responses


Multimodal Alexa Skill allows for Alexa skills in both audio and visual responses to user's requests.


Handlers and Alexa’s SDK


Handlers are used to communicate requests from Intents with Alexa Skills Development SDK. The intents intake the utterances from the end-users that see if any slots are mapped to that intent. Then take that input to the handlers and relay that to the Alexa Skills Development SDK.


Skill Types and Their API’s

There are many different skill types and each has a different API they call for that functionality.


Smart Home Skill


Smart Home Skill control devices in your home. This is done by connecting the Smart Home Skill APIs, Smart Home Energy Skills, Smart Home Security Skills, and Multi-capability Skills. Examples of objects that it controls and interacts with are cameras, lights, locks, thermostats, cameras, and security systems.

3) Automotive Skills are skills that are meant to integrate with the user's car. There are two different types of functionality in this.

a) Driving Skills allow users to be productive while driving so that they can keep an eye on the road.

b) Connected Vehicle Skills Alexa to control your car functionality like turning on your car.

4) Alexa Skills for Business, Hospitality, and Residential

a) Alexa for Business: Alexa business skills can deeply integrate with a business infrastructure to help the workload of the business by automating tasks. Like checking to see if there are any meeting rooms available and then scheduling that meeting with the booking API. Similarly, the conferencing skill allows employees to control the room lights, the volume of speakers, etc.

b) Alexa Hospitality uses technology to integrates to deploy and manage functionality in hotels, resorts, vacation rental communities, hospitals, etc. The Alexa Hospitality functionality is rather extensive for how it integrates. It uses the Property, Room Type, and Room API, Role API, Unit Settings API, Endpoint API, Device Group Management API, Discovery Session API, Skill Management API, Communications API, and Notifications API.
